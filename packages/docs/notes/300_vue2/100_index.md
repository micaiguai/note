# 简介
  现代前端框架
## 历史背景
  纯Html -> Jsp -> JQuery -> Vue
## 作用
### MVVM
  - Model：模型层，处理数据逻辑
  - View：视图层，展示UI
  - ViewModel：视图模型层，Model和View的通信桥梁
### 组件化
  组件化是将HTML、JS、CSS抽象出来，作用有：
  1. 低耦合
  2. 调试方便
  3. 提高可维护性
### 指令
  - v-if：   是否渲染
  - v-for：  列表渲染
  - v-bind： 属性绑定
  - v-on：   事件绑定
  - v-model：双向数据绑定
### Vue vs JQuery
  - 操作DOM：Vue可以不用关注，JQuery需要频繁操作
