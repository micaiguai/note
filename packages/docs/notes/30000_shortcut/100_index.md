# shortcut

## vscode
- `opt` + `shift` + `↑` / `↓`: 复制框选内容
- `cmd` + `opt` + `[` / `]`: 折叠 / 展开
- `opt` + `z`: 切换换行

## macos
- `cmd` + `F3`: 展示桌面
- `ctrl` + `cmd` + `Q`: 锁屏
- `opt` + `cmd` + `D`: dock show/hide
- `cmd` + `ctrl` + `space`: symbol panel

## idea
- `ctrl` + `alt` + `V`: introduce local variable
