# shortcut

## vscode
- `option` + `shift` + `↑` / `↓`: 复制框选内容
- `command` + `option` + `[` / `]`: 折叠 / 展开
- `option` + `z`: 切换换行

## macos
- `command` + `F3`: 展示桌面
- `ctrl` + `command` + `Q`: 锁屏
- `fn` + `ctrl` + `F3`: dock show/hide
- `command` + `ctrl` + `space`: symbol panel

## idea
- `ctrl` + `alt` + `V`: introduce local variable
